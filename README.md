# 🚍 NoQ – No Need of Queue

**NoQ** is a smart digital transport card management system designed to eliminate long queues at bus and metro recharge centres.  
It enables commuters to recharge, manage, and discontinue their transport cards seamlessly through a secure mobile/web interface.

---

# 📌 Project Overview
Public transport users often face long queues and manual processes at recharge centres to renew or top-up their travel cards. This leads to time consumption, inconvenience, and dependency on physical locations.

**NoQ** solves this problem by digitizing the entire recharge and card management process, allowing users to manage their transport cards anytime and from anywhere.

---

# ❗ Problem Statement
Traditional transport card recharge systems rely on physical centres where:
- Users must stand in long queues
- Manual data entry and payment processing occurs
- Periodic visits are required for top-ups
- No centralized digital solution exists

This results in inefficiency, time wastage, and poor user experience.

---

# 💡 Our Solution — NoQ
**NoQ** provides a digital platform where users can:

- Login using registered mobile number
- Verify securely using OTP
- View linked transport card details
- Recharge card remotely
- Receive student discount automatically
- Generate digital payment receipt
- Discontinue card securely with OTP verification

All without visiting a physical recharge centre.

---

# ✨ Key Features
- 📱 Mobile number based secure login with OTP  
- 🎫 Smart card dashboard with trip tracking  
- 💳 Recharge options (1 Month – 60 trips, 3 Months – 180 trips)  
- 🎓 Automatic student discount calculation  
- 🧾 Digital payment receipt with cost breakdown  
- 🔐 Activation code verification for secure top-up  
- 🚫 Card discontinuation with 6-digit OTP confirmation  
- 🎨 Clean and user-friendly interface  
- ⚡ Fast and scalable digital workflow

# 🔄 System Workflow
1. User logs in using registered mobile number  
2. OTP verification ensures secure authentication  
3. Dashboard displays linked card details and trips  
4. User selects recharge plan and enters activation code  
5. System calculates fare and student discount  
6. Digital payment receipt generated  
7. Card updated with new trips  
8. Optional card discontinuation with OTP verification
